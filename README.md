# Nightreign Patterns

Nightreign Patterns aide à retrouver rapidement une carte possible d'Elden Ring Nightreign à partir des informations visibles en jeu.

## Fonctionnalités

- Filtrer les patterns par Nightlord, type de carte et point de spawn.
- Afficher directement la carte associée à un pattern.
- Revenir à une carte vierge pour choisir un point de spawn.
- Voir uniquement les points de spawn disponibles pour le type de carte sélectionné.
- Distinguer les cartes Great Hollow de la carte standard.
- Identifier rapidement l'événement indiqué sur les patterns quand il est connu.

## Utilisation

1. Sélectionner un Nightlord si nécessaire.
2. Choisir le type de carte.
3. Sélectionner un spawn ou laisser tous les spawns visibles.
4. Cliquer sur une pattern dans la liste de droite pour afficher sa carte au centre.
5. Utiliser le bouton carte vierge à côté du spawn pour revenir à la sélection des spawns.

## Notes

- Les spawns Great Hollow sont séparés des autres spawns.
- Le bouton Réinitialiser remet les filtres à zéro et réaffiche la carte vierge.
- Les icônes de Nightlord servent uniquement à faciliter l'identification visuelle.
